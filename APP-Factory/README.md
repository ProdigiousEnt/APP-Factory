# App Support Pages - GitHub Pages Setup

## Quick Setup Instructions

### 1. Create GitHub Repository

1. Go to [GitHub](https://github.com)
2. Click "New Repository"
3. Name it: `yourusername.github.io` (replace `yourusername` with your GitHub username)
4. Make it **Public**
5. Click "Create Repository"

### 2. Upload These Files

Upload these files to your repository:
- `index.html` - Main landing page
- `splitsmart.html` - SplitSmart support page

### 3. Enable GitHub Pages

1. Go to repository Settings
2. Click "Pages" in left sidebar
3. Under "Source", select "main" branch
4. Click "Save"

### 4. Update Email Address

In both HTML files, replace `support@example.com` with your actual support email.

### 5. Your URLs

After setup, your pages will be available at:
- Main page: `https://yourusername.github.io/`
- SplitSmart: `https://yourusername.github.io/splitsmart.html`

**Use the SplitSmart URL as your Support URL in App Store Connect!**

## Adding More Apps

To add support for another app:

1. Copy `splitsmart.html`
2. Rename it (e.g., `zensynth.html`)
3. Update the content (app name, icon, FAQs, etc.)
4. Add a card to `index.html` linking to the new page
